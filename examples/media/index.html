Flash Vars
logLevel Уровень логирования для дебаг-консоли
publisherRect Прямоугольная область для размещения публикуемого видео - x,y,width,height. Значение по умолчанию - "0,216,88,72".
subscriberRect Прямоугольная область для размещения просматриваемого видео - x,y,width,height. Значение по умолчанию -"0,0,352,288". Минимальный размер: width: 215, height: 138
camFPS Частота кадров камеры. Значение по умолчанию - 10.
camQualityBW Ширина видеопотока камеры в байт/сек. Значение по умолчанию - 0.
camQuality Качество изображения камеры. Значение по умолчанию - 80.
camKeyFrame Интервал ключевых фреймов камеры. Значение по умолчанию - 48.
camWidth Ширина изображения камеры. Значение по умолчанию - 352.
camHeight Высота изображения камеры. Значение по умолчанию - 288.
camMotionLevel Уровень детектирования движения камеры. Значение по умолчанию - 0.
camMotionTimeout Таймаут детектирования движения камеры в миллисекундах. Значение по умолчанию - 20000.
camCodec Кодек камеры. Возможные значения - "sorenson|h264". Значение по умолчанию - "sorenson".
micSilenceLevel Уровень детектирования активности микрофона. Значение по умолчанию - 0.
micSilenceTimeout Таймаут детектирования активности микрофона в миллисекундах. Значение по умолчанию - 20000.
micCodec Кодек микрофона. Возможные значения - "speex|nellymoser|pcmu|pcma". Значение по умолчанию - "nellymoser".
micGain Усиление микрофона. Диапазон - 0-100. Значение по умолчанию - 50.
micRate Частота дискретизации микрофона в кГц. Значение по умолчанию - 8.
bandwidthEndpoint URL для детектора оптимизации качества QOS. Как правило совпадает с URL приложения live. Если значение не задано - QOS использоваться не будет.
bandwidthCheckInterval Интервал в миллисекундах для измерения ширины канала QOS. Значение по умолчанию - 5000.
bandwidthDisableDownloadCheck Игнорировать измерение полосы download (от сервера к клиенту). Значения - 'true|false'. Значение по умолчанию - false.
bandwidthDisableUploadCheck Игнорировать измерение полосы upload (от клиента к серверу). Значения - 'true|false'. Значение по умолчанию - false.
bandwidthMinCameraLimit Минимальная полоса пропускания в кБит/сек для камеры. Если ширина полосы будет ниже - камера будет отключена. Значение по умолчанию - 100.
callTimerX Позиция текстового поля таймера. Значение по умолчанию - 0.
callTimerY Позиция текстового поля таймера. Значение по умолчанию - 0.
callTimerFontColor Цвет текстового поля таймера. Значение по умолчанию - 0.
callTimerFontSize Размер текстового поля таймера. Значение по умолчанию - 10.
clickTag URL ссылки для фонового изображения. См. loadImage().
